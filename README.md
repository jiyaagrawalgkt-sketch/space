# 🚀 Space

An interactive and visually engaging **space-themed website** developed using **HTML, CSS, and JavaScript**. The project is designed to provide users with an immersive experience by showcasing the beauty of space through modern web design, smooth animations, and responsive layouts.

Whether viewed on a desktop or a mobile device, the website offers an attractive interface and an enjoyable browsing experience while demonstrating core front-end web development skills.

##  Features

*  Modern and attractive space-inspired design
*  Fully responsive layout for all screen sizes
*  Interactive elements powered by JavaScript
*  Smooth animations and stylish user interface
*  Fast and lightweight website
*  Clean and organized code structure

##  Technologies Used

* **HTML5** – Structure of the website
* **CSS3** – Styling, layout, and animations
* **JavaScript** – Interactivity and dynamic functionality

##  Project Structure

```
space/
│── index.html
│── style.css
│── script.js
└── README.md
```

##  Purpose of the Project

The goal of this project is to strengthen front-end web development skills by building an interactive website using core web technologies. It demonstrates responsive design principles, creative UI development, and JavaScript-based interactivity.

##  How to Run

1. Clone this repository.
2. Open the project folder.
3. Launch `index.html` in your preferred web browser.
